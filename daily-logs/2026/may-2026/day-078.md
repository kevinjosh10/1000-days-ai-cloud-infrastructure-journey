### 2️⃣ Learned CIDR Block Fundamentals

Studied CIDR notation and IP allocation.

Example CIDR:

```text
10.0.0.0/16
```

Understood:
- CIDR defines the IP range of the VPC
- `/16` determines network size
- Larger CIDR blocks provide more IP addresses

Learned common subnet sizes:

| CIDR | Available IPs |
|---|---|
| /16 | 65,536 |
| /24 | 256 |
| /28 | 16 |

Learned:
- Smaller CIDR suffix = larger network
- CIDR planning is important in production environments

Visualized:

```text
10.0.0.0/16
      ↓
Large Private Network
```

### 7️⃣ Learned Public vs Private Subnet Architecture

Studied subnet exposure concepts.

#### Public Subnets

Used for:
- Load balancers
- Bastion hosts
- Public web servers
- NAT gateways

Characteristics:
- Internet accessible
- Public IP assignment possible

#### Private Subnets

Used for:
- Databases
- Internal APIs
- Backend services
- AI workloads

Characteristics:
- No direct internet access
- Internal communication only

Visualized:

```text
Internet
    ↓
Public Subnets
    ↓
Private Subnets
```

Learned:
- Sensitive systems should remain private
- Layered networking improves security
