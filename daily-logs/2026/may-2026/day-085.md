### 3️⃣ Learned About the AWS Root Account

Studied AWS root account behavior and risks.

Understood:
- Root account has unrestricted AWS access
- Root credentials should never be used for daily work
- Root account is the owner of the AWS account

Learned:
- Root access bypasses normal IAM restrictions

Visualized:

```text

        ↓
```

Learned:
- Root account compromise is extremely dangerous

Studied AWS recommendations for securing root credentials.

- Avoid daily usage of root account

Understood:
- Root account should only be used for emergency account-level tasks

```text
Root Account
     ↓
```
