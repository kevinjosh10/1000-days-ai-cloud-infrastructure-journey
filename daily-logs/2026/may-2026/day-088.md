### 8️⃣ Created MFA Enforcement Policy

Created an IAM policy denying all actions when MFA is absent.

Policy created:

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "DenyAllIfNoMFA",
      "Effect": "Deny",
      "Action": "*",
      "Resource": "*",
      "Condition": {
        "BoolIfExists": {
          "aws:MultiFactorAuthPresent": "false"
        }
      }
    }
  ]
}
```

Visualized:

```text
MFA Missing
      ↓
AWS Access Blocked
```

Learned:
- MFA enforcement policies provide strong AWS account protection
