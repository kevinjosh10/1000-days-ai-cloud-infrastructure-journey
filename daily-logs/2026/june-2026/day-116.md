### 1️⃣ Learned About Amazon EFS

Studied how AWS provides a fully managed shared file system.

Understood:

- Fully managed by AWS
- Supports shared access
- Automatically scales

Visualized:

```text

    \
     EFS

   /
```

Understood:

```text

 ↓

 ↓

```

Visualized:

```text

EFS → Many Servers
```

Learned:
- EBS is block storage for a single EC2 instance
- EFS is shared storage for multiple EC2 instances
