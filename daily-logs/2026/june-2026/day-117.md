### 1️⃣ Learned About AWS Storage Gateway

Studied how AWS connects on-premises storage with AWS Cloud.

Understood:

- Storage Gateway is a hybrid cloud storage service
- Connects on-premises environments to AWS
- Allows existing applications to continue working
- Integrates with AWS storage services

Visualized:

```text
On-Premises
      ↓
```
