### 1️⃣1️⃣ Studied Synchronous Invocations

Learned request-response execution.

Understood:

- Caller waits for response
- Immediate result returned
- Common for APIs

Visualized:

```text
Client
  ↓
Lambda
  ↓
Response
```

Learned:

- API Gateway commonly invokes Lambda synchronously.

---

### 1️⃣2️⃣ Studied Asynchronous Invocations

Learned event-driven execution.

Understood:

- Caller does not wait
- Lambda processes event later
- Built-in retries

Visualized:

```text
Event
  ↓
Lambda
```

Learned:

- S3, SNS and EventBridge commonly invoke Lambda asynchronously.

### 1️⃣3️⃣ Studied Poll-Based Invocations

Visualized:

