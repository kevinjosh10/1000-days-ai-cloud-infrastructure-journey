### 1️⃣ Learned About Amazon S3 Versioning

Studied how AWS protects data using object versioning.

- Every object can have multiple versions
- Previous versions remain accessible
- Accidental overwrites can be recovered

Visualized:

```text

   ↓
Version 1
Version 2
Version 3
```

- Versioning provides built-in protection against accidental changes


- Older versions remain recoverable

```text

     ↓

v2
v3
```

Learned:
- S3 behaves like file version control for object storage

### 5️⃣ Learned About Delete Markers

Studied how deletion works in versioned buckets.

Understood:
- Objects are not immediately deleted
- AWS creates a delete marker
- Previous versions remain stored

Visualized:

```text

      ↓
```

- Deleting an object does not remove its data when versioning is enabled
